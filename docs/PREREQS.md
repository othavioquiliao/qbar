# Prerequisites

## System
- Linux + Waybar
- `bash`, `jq`, `curl`, `python3`, `lsof`

## Claude Code
- `claude` CLI installed
- Login happens in CLI (`/login`)

## Codex
- `codex` CLI installed (bun or npm)
- Login via CLI

## Antigravity
- Antigravity IDE installed
- IDE running for local LSP
- Optional cloud fallback via `antigravity-usage`
